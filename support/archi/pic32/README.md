# PIC 32 bits support

This architecture support file handle all 32 bits families of Microchip. compared to 16 bits, 32 bits range offers various advantages :

* C++ compiler support
* larger memory addressing
* code relocalisation
* execution from RAM

(316 devices on January 2023)
